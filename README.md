## 📥 Clone/Baixe

1. Clone:

    ### Clonando

    ```bash
    git clone git@github.com:cleibp/basic-pascal.git
    ```
1. Download:
    ### Download Manual

        1.  Download do repositório
        2.  Descompacte para o diretório desejado


1. Comandos Básicos de MKDOCS

    ### Verificar versão
      ```
      mkdocs --version

      ```
    ### Criando o projeto
      ```
      mkdocs new my-project

      ```
    ### Entrando no projeto
      ```
      cd my-project

      ```
    ### Rodando o projeto
      ```
      mkdocs serve

      ```
    ### Gerando o build do projeto
      ```
      mkdocs build
      ```

    ### Rodando a aplicação
      ```
      mkdocs serve
      ```
