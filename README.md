# baseOfLinks
Documentação de links úteis no processo de desenvolvimento utilizando a ferramenta mkdocs.

# Comandos Básicos

## Verificar versão
```
mkdocs --version

```
## Criando o projeto
```
mkdocs new my-project

```
## Entrando no projeto
```
cd my-project

```
## Rodando o projeto
```
mkdocs serve

```
## Gerando o build do projeto
```
mkdocs build
```

# Rodando a aplicação
```
mkdocs serve
```
